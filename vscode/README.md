# VS Code My CheatSheet

## Shortcut for macOS

### General

|  Keybinding  |  When  |  Note |
| ---- | ---- | ----- |
|  ⌘+P  |  Quick Open, Go to File…  | ファイルを検索して開く |

### Terminal

|  Keybinding  |  When  |  Note |
| ---- | ---- | ----- |
|  ⌘+J  |  Focus on the terminal  | ターミナルへフォーカスする |
